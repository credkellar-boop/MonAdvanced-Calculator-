# Ultimate Advanced Calculator

A high-performance, infinite-precision computational engine designed for sovereign-level architecture and advanced mathematical processing. 

## Capabilities
* **Symbolic Math:** Exact fractions, algebraic simplification, and trigonometric identities.
* **Calculus Engine:** Automated derivatives, indefinite and definite integrals, and limit evaluations.
* **Arbitrary Precision:** Configurable decimal precision capable of managing calculations at the decillion scale without floating-point rounding errors.
* **REST API Ready:** Built with FastAPI endpoints for seamless integration into larger data networks or trading algorithms.

## Quickstart
1. `pip install -r requirements.txt`
2. Run the core engine tests: `pytest tests/`
3. Start the API server: `uvicorn src.api.routes:app --reload`
